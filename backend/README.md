# 苏小问-问卷系统 后端代码


## 简介

* 基于 RuoYi-Vue-fast
* 后端技术栈：Spring Boot、Spring Security、Redis & Jwt。
* 权限认证使用Jwt，支持多终端认证系统。
* 加载动态权限菜单，多方式轻松权限控制。

## 配置

请参考： https://www.cnblogs.com/liuyangfirst/p/15998984.html

- 请手动替换有关配置文件中的数据库、Redis等配置
